✨ Quick Start Guide
  • 1. Connect Channels: Link your Facebook and Instagram profiles below under settings.
  • 2. Map Voice Guidelines: Define your brand's unique personality in the prompt box below. You can command the system using specific rules, such as:
    • Tone & Vibe: "Write in a witty, sarcastic tone" or "Keep it highly professional and corporate."
    • Formatting Rules: "Never use emojis," "Always write in short, punchy single sentences," or "Use bullet points."
    • Hashtags & Structure: "Always end the post with a call to action and exactly 5 relevant hashtags."
  • 3. Process Assets: Load an image file to extract visual context and generate tailored draft variations.
  • 4. Polish & Sync: Select a card to fill the text pane, make any final edits, and push directly live.
